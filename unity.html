Unity assignment

Unity is een programma waarin je een virtuele wereld kan maken. In Unity worden twee soorten projecten ge creëerd. Games en simulaties. Door middel van deze assignments ga ik zelf een interactieve simulatie bouwen.

assignment 1

De eerste opdracht was om een nieuw terrein te maken met een vloer. Ook moest ik door middel van een kwast heuvels maken op de vloer. Dit zorgde voor een eigen creatie van een landschap.

assignment 2

Bij de tweede opdracht moest ik een player creëeren. Door deze player kan je zelf door je landschap lopen.

assignment 3

De derde opdracht hield in dat ik objecten moest toevoegen aan de virtuele wereld. De objecten moesten natuurkunde eigenschappen hebben. Zoals zwartekracht bijvoorbeeld. Ik had een pyramide gemaakt die niet helemaal goed op elkaar stond, zodat ze zouden omvallen.

assignment 4

Bij de vierde opdracht moest ik een object uit de asset store in mijn wereld plaatsen. Ik heb ervoor gekozen om een tijger aan mijn wereld toe te voegen.

assignment 5

De vijfde opdracht was een grote opdracht. Ik moest een trigger toevoegen aan de tijger. Zodra ik dichtbij mijn tijger zou komen, zou er een vuurtje ontstaan boven de tijger.

assignment 6

Bij de laatste opdracht moest ik het programma exporteren. Dat zorgde ervoor dat je dat je een echte applicatie hebt die je kunt gebruiken zonder dat je Unity aan moet zetten.

Madmapper assignment

Madmapper is een gebruiksvriendelijke tool voor 3D projection mapping. Wat heel handig is aan Madmapper is dat het samen kan werken met Siphon. Helaas werkt dit alleen voor apple, daarom heb ik de opdracht samen gedaan met Zuhal, Fleur, Natalia en Giulia. Door middel van processing en Syphon, kun je een interactive omgeving creëeren. Je kunt daarmee iets projecteren op een physiek object.

Voor deze opdracht moesten we uitvinden hoe madmapper werkt en er een projectie meemaken op een speciaal oppervlak. Dit kon een doos, een muur of een persoon zijn. Wij hebben ervoor gekozen om een doos te gebruiken. De filmpjes zijn gemaakt door Zuhal Özcan.
