Computer vision

Computer vision is de naam van de oefening die ik heb gedaan. Daar gebruik je het programma 'processing' voor. Processing is een tool voor mensen die interactieve afbeeldingen en animaties willen maken. Het programma wordt gebruikt door studenten, artiesten, ontwerpers, uitvinders en hobbyisten. Het is een digitaal schetsboek.

assignment 1

De eerste opdracht was om drie ovalen te maken. Alle drie een andere vorm, kleur en positie. Dit was een simpele opdracht.

assignment 2

Bij de tweede opdracht moest ik bijna hetzelfde doen als de eerste opdracht. Alleen moest ik bij deze opdracht drie verschillende vormen maken. Dit was ook een simpele opdracht, alleen moest ik wel goed nadenken over de co√∂rdinaten van de hoeken van de figuren.

assignment 3

Bij de derde opdracht moest ik twee verschillende figuren maken, met verschillende kleuren. Ook moesten ze interactief zijn. Ik heb ervoor gekozen om een lijn en een cirkel te maken. De lijn stond met een uiteinde vast in het midden en het andere punt volgde de muis. De cirkel volgde ook de muis en de kleur van de cirkel veranderde op basis van de locatie van de muis. Deze opdracht vond ik ook nog vrij simpel.

assignment CV1

Bij deze opdracht moest ik eerst code downloaden van blackboard. Deze code liet een afbeelding van een man zien met een groen vierkant om zijn gezicht. Ik moest in plaats van de foto, mijn webcam gebruiken zodat je mijn gezicht live ziet. Daarna moest ik het groene vierkant in een cirkel veranderen.

assignment CV2

Bij deze opdracht moest ik de code van de vorige opdracht zo aanpassen zodat ik een rode clownsneus kreeg en een zwarte balk voor mijn ogen. De neus is me gelukt. De balk lukte niet als een geheel, maar wel twee vierkantje voor de ogen. De computer was niet heel nauwkeurig. Hij zag veel dingen voor ogen en neuzen aan. Dit maakte de screenshot maken vrij lastig.

assignment CV3

Bij de laatste opdracht moest ik een code downloaden van blackboard. Deze code moest ik dan aanpassen tot er iets leuks gebeurde. Ik heb dit meerdere malen geprobeert, maar het is me helaas niet gelukt.
